# AtCoder 介绍
日本的一个 OI 网站，网址为 [AtCoder](https://atcoder.jp/)。基本每周都会有一个在线竞赛，一般是六题，A ~ F。  
竞赛难度分为：  
AtCoder Beginner Contest。简称 ABC，入门赛。难度和国内 NOIP 普及组类似，比普及组 T4 略微低一点。  
AtCoder Regular Contest。简称 ARC。  
AtCoder Grand Contest。简称 AGC，大师赛。  
还会有一些企业竞赛。
